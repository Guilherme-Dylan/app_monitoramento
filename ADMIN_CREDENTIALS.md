## 🔐 Como Fazer Login

1. Acesse a aplicação no navegador
2. Clique no botão **"Entrar"** no canto superior direito
3. Você será redirecionado para o **Manus OAuth**
4. Faça login com suas credenciais Manus
5. Você será automaticamente sincronizado como administrador

---

## 📋 Funcionalidades de Administrador

Com a conta de administrador, você terá acesso a:

- ✅ **Painel Administrativo**: Visualizar e gerenciar todas as solicitações de busca
- ✅ **Gerenciamento de Denúncias**: Visualizar e atualizar status de denúncias anônimas
- ✅ **Alteração de Status**: Aprovar ou rejeitar solicitações
- ✅ **Geração de Relatórios**: Baixar relatórios em PDF com histórico completo
- ✅ **Dashboard**: Acompanhar métricas e estatísticas do sistema

---

## ⚠️ Observações Importantes

1. **Autenticação OAuth**: O sistema usa Manus OAuth para autenticação. Os usuários administradores foram criados no banco de dados, mas o login é feito através do Manus OAuth.

2. **Sincronização Automática**: Quando você faz login via Manus OAuth, o sistema sincroniza automaticamente suas informações com o banco de dados.

3. **Segurança**: Nunca compartilhe suas credenciais de acesso. Os OpenIDs são únicos e identificam cada usuário no sistema.

4. **Primeiro Acesso**: Na primeira vez que você fizer login, o sistema criará automaticamente sua sessão como administrador.
